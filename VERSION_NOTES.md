# 版本说明

## v0.9.2 · 2026-07-22

- 新增 GitHub Pages 公开发布入口，线上版仍是无服务器、无数据库依赖的纯浏览器单文件网站。
- 新增 `npm run build:pages`，生成 `pages-dist/index.html` 和 `.nojekyll`，并校验其内容与本地离线 HTML 完全一致。
- 离线打包固定 esbuild 版本并规范化其依赖路径，消除 pnpm/npm 与 macOS/Linux 解析出的本机绝对路径差异，使本地和 Pages 构建可按字节复核。
- 新增 GitHub Actions 工作流：使用 Node.js 22 生成数据、执行生产构建、运行页面/data 与多机避障回归测试，测试通过后才部署 Pages。
- README 和使用说明补充线上入口、离线使用、外部文件本地读取、仿真数据边界、无开源许可证及“不控制真实无人机”声明。
- 公开发布没有改变实时代理核心逻辑、世界坐标、三维航迹、实验回放、阶段 4A/4B/4C 或多机防卡死避障逻辑。

## v0.9.1 · 2026-07-20

- 补充地图图层、遥测字段、决策动作、统计指标和端侧基准的高低含义与组合判断方法。
- 覆盖五组实验的27张统计图、95%置信区间、热图颜色、single-seed代表案例和阶段4C双地图对照的正确读法。
- README增加用户指南入口；网站功能逻辑和实时代理核心逻辑未改变。
